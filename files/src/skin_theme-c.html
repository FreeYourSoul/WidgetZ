wz_init_skin_theme

void wz_init_skin_theme(struct WZ_SKIN_THEME *theme)

Initializes a skin theme.  Do not pass wz_skin_theme to this function, but make a copy of it first.  You should initialize the bitmap variables inside the WZ_SKIN_THEME struct before calling this function.  Don’t forget to call wz_destroy_skin_theme on the theme when you are done with it.

wz_destroy_skin_theme

void wz_destroy_skin_theme(struct WZ_SKIN_THEME *theme)

Destroys a skin theme that has been initialized by wz_init_skin_theme.  Note that it does not free the passed pointer.
