wz_send_event

int wz_send_event(WZ_WIDGET *wgt,
const ALLEGRO_EVENT *event)

Sends an event to a widget, and to any of its focused children, propagating down the tree.  The first widget that processes the event breaks the process.  Note that the first widget gets the event whether it is focused or not.

Returns

1 if the event was handled by some widget, 0 if it was not

wz_trigger

void wz_trigger(WZ_WIDGET *wgt)

Triggers a widget.  What the widget does depends on the widget.  Generally, it will send an event that is characteristic of the events that it usually sends.  For example, triggering a button will simulate an impression of the button, causing it to send WZ_BUTTON_PRESSED event.

Parameters

scaleScaling factor.  0 results in a black color, 1 results in no change
