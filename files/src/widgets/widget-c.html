wz_create_widget

WZ_WIDGET* wz_create_widget(WZ_WIDGET *parent,
float x,
float y,
int id)

Creates a standard widget.  You generally don’t need to use this, unless you want a do-nothing dummy widget.

Parameters

idThe id of this widget, which will be used to identify it when it sends events to the user’s queue.  Pass -1 to automatically assign an id.  If automatically assigned, id will equal the id of its previous sibling plus one, or if it has none, the id of its parent plus one.  If it has no parent, its id will be 0.
